Javascript-on-Journey
=====================

javascript toolbox for webapp.

![Journey Generations](//raw.github.com/iolo/jj/master/app/img/journey-generations.jpg)

getting started
---------------

```
$ npm install
$ grunt sever
$ open http://localhost:8000
```
